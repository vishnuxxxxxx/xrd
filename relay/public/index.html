Who's behind the dashboard

RELAY started as a side project between a handful of community managers who kept seeing the same thing: small brands and creators getting buried by DMs, comment storms, and platform changes they didn't have time to track. We pulled that work into one rotating team instead.

Today RELAY staffs shifts across eight platforms, hands off cleanly between time zones, and keeps a public roster so anyone working with us can see who's actually on right now — no guessing whether a message landed with a real person.

Every member listed below has been through ID verification and onboarding before going on shift. You can check anyone's membership status yourself in the ID Check section.
